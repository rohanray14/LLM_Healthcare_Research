To enable Google Sheets integration, set these environment variables on Render:

GOOGLE_SHEETS_CREDENTIALS_JSON The full JSON content of your Google service account key file
GOOGLE_SHEET_ID The spreadsheet ID from the Google Sheet URL
e.g. from https://docs.google.com/spreadsheets/d/SHEET_ID_HERE/edit
Setup Steps
  1. Go to Google Cloud Console and enable the Google Sheets API
  2. Go to IAM & Admin > Service Accounts and create a service account
  3. Create a JSON key for the service account and download it
  4. Share your Google Sheet with the service account email (the client_email in the JSON)
  5. On Render, add GOOGLE_SHEETS_CREDENTIALS_JSON with the full JSON content and GOOGLE_SHEET_ID with your sheet ID
  6. Redeploy the app
